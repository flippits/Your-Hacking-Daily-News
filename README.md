# Your Hacking Daily News

![Daily Hacking Digest](https://github.com/flippits/Your-Hacking-Daily-News/actions/workflows/daily.yml/badge.svg)
![Daily Issue](https://img.shields.io/badge/daily-issue-green)
![Weekly Recap](https://img.shields.io/badge/weekly-recap-blue)

Welcome to a daily, magazine‑style digest of hacking and cybersecurity news, advisories, and research highlights.

**Latest issue:** See the newest folder in `News/` (each issue is a `README.md`)  
Each day a new dated issue is published in Markdown, ready to read like a quick morning magazine.

**Weekly recap:** Every Sunday a `Weekly-YYYY-WW/README.md` issue is generated in `News/`.

**Magazine site:** Enable GitHub Pages from `/docs` to publish a clean front page.
